Precedencia

1 - Operadores aritimeticos
2 - Operadores Relacionais

Relacionais

5 > 2 = true
7 < 4 = false
8 >= 8 = true
9 <= 7 = false
5 == 5 = true
4 != 4 = false

Identidade

5 == 5 = true

5 == '5' = true (o sinal de = ele nao testa o tipo ele testa o valor os cincos tem a mesma grandeza por isso que a resposta e true)

5 === '5' = false (testando se os 5 sao identicos se sao o mesmo valor e do mesmo tipo)

5 === 5 = true

Lógicos

! negação
&& conjuunção
|| disjunção

Negação

[!] Esse e tratado como operador unario (so tem um operando) ou true ou false uma coisa que nao e verdadeira e falsa e uma coisa que nao e falsa e verdadeira.

[!] true = false
[!] false = true

Conjunção

[&&] Esse e um operador binario ou seja eu tenho eu tenho dois valores logicos que vao me dar um resultado logico ex. (Eu quero uma caneta azul e outra vermelha eu consegui as duas entao ficarei satisfeito se eu conseguir uma tambem fico insatisfeito se nao consegui nenhuma tbm fico insatisfeito) so e true se as duas for verdadeira

true [&&] true = true
true [&&] false = false
false [&&] true = false
false [&&] false =false

Disjunção

[||] Esse e um operador binario nesse operador basta que um deles seja verdadeiro para que o resultado seja verdadeiro

true [||] true = true
true [||] false = true
false [||] true = true
false [||] false = false

idade >= 15 && idade <= 17 (A idade esta entre 15 e 17)
estado == 'RJ' || estado == 'SP' (O estado e RJ ou SP)
salario > 1500 && sexo != 'M' (O salario e acima de 1500 e nao é homem?)

teste [?] true [:] false
